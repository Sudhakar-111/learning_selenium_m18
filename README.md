# Selenium Automation Project

## Description
This project is a Selenium automation framework using Java and TestNG.

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- Maven

## Features
- Data-driven testing using Excel
- Reusable utility classes
- Screenshot capture

## Project Structure
- src/main/java → utility classes  
- src/test/java → test scripts  
- pom.xml → Maven file  

## How to Run
1. Open project in Eclipse
2. Run testng.xml file
